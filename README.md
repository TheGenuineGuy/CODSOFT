This repository contains the tasks completed as part of the **CodSoft Python Programming Internship**. Each task is built as a standalone Python GUI application using `customtkinter`, with data persisted locally in JSON.

## Tasks Completed

### Task 1 — To-Do List
A GUI-based to-do list application built with `customtkinter`.
- Add tasks via input field or Enter key
- Mark tasks complete/incomplete with checkboxes
- Delete tasks
- Tasks persist automatically to `todo.json`

**Run:**
```bash
cd Task1_Todo
python main.py
```

### Task 2 — Contact Book
A contact management application built with `customtkinter`.
- Add contacts with name, phone, and email
- View all saved contacts in a scrollable list
- Update existing contact details
- Delete contacts
- Data persists to `contacts.json`

**Run:**
```bash
cd Task2_ContactBook
python main.py
```

### Task 3 — Password Generator
A customizable password generator built with `customtkinter`.
- Specify desired password length
- Toggle character sets: letters, numbers, symbols
- Generate and copy password to clipboard

**Run:**
```bash
cd Task3_PasswordGenerator
python main.py
```

## Tech Stack
- Python 3
- customtkinter (GUI)
- json (data persistence)

## Setup
```bash
pip install customtkinter
```

Each task folder contains its own `main.py`, a JSON data file for storage, and a `screenshots/` folder showing the app in use.
